# Kevin Lam Seck

**Full-stack developer building thoughtful software, developer tools, and AI-powered workflows.**
I care about systems that are fast, maintainable, and genuinely useful, from product-facing apps to local-first tooling, architecture, and automation.

## About

I like turning ideas into solid products with a strong technical foundation. My work sits at the intersection of **product thinking**, **software architecture**, and **developer experience**.

Over time, I’ve grown especially interested in:

- system design and scalable architecture
- AI-assisted workflows and agent orchestration
- developer tooling and local-first software
- building products that stay simple for users and clean for teams

## What I’m working on

- **Developer tooling** around code intelligence, search, and local-first workflows
- **AI-native products** that combine useful automation with strong engineering foundations
- **Desktop experiences** with modern full-stack patterns using TypeScript, React, Rust, and Tauri
- **Full-stack apps** designed to move fast without compromising maintainability
- **Architecture-driven builds** with a focus on modularity, clear boundaries, and long-term evolvability

## Selected projects

- **[kensho](https://github.com/KevinLamSeck/)** — a privacy-first code intelligence project with a specialized search agent, combining embeddings, BM25, and deeper codebase exploration patterns for augmented developer workflows
- **[quickrack](https://github.com/KevinLamSeck/)** — an inventory web app
- **[mavory](https://github.com/KevinLamSeck/)** — a cross-platform meeting assistant app built with Tauri, designed around desktop productivity and modern full-stack patterns
